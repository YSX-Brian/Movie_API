Movie API

This API is an application made using Node.js and Express. It will store data on a variety of different movies, directors, and genres. Users will be able to create an account, add their personal info, and create a list of their favorite movies.

Description URL HTTP Method Request Format Response Format
Return a list of ALL movies to the user /movies GET None All movies as objects within a JSON array.
Return data about a single movie by title to the user /movies/:title GET None A JSON object which details the requested movie.
Return data about a genre by name/title /movies/genre/:genreName GET None A JSON object which details the requested genre.
Return data about a director by name /movies/directors/:directorName GET None A JSON object which details the requested director.
Allow new users to register /users POST A JSON object with a name and array of favorite movies. Ex: {"Username": "", "FavoriteMovies": []} The same request JSON object with a unique ID added + confirmation status code.
Allow users to update their username /users/:id PUT A JSON object with a new name. Ex: {"Username": ""} A JSON object reflecting the updated name.
Allow users to add a movie to their list of favorites /users/:id/:movieTitle POST None A message confirming the addition of a movie.
Allow users to remove a movie from their list of favorites /users/:id/:movieTitle DELETE None A message confirming the removal of a movie.
Allow existing users to deregister /users/:id DELETE None A message confirming successful deregistration.
