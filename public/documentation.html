Movie API

This is an application made using Node.js and Express. Access a variety of different movies, directors, and genres. Users will be able to create an account, add their personal info, and create a list of their favorite movies.

Description URL HTTP Method Request Format Response Format
Return a list of all movies. /movies GET None All movies as objects within a JSON array.
Return data about a single movie by title to the user /movies/:Title GET None A JSON object which details the requested movie.
Return data about a genre by name/title /movies/genre/:GenreName GET None A JSON object which details the requested genre. Ex:
{"Name": "Horror",
"Description": "Horror is a film genre that seeks to elicit fear or disgust in its audience for entertainment purposes."}
Return data about a director by name /movies/directors/:DirectorName GET None A JSON object which details the requested director. Ex:
{"Name": "Hayao Miyazaki",
"Bio": "Director Bio"}
Allow new users to register /users POST A JSON object with at minimum a username, password, and email address. Ex:
{"Username": String (required),
"Password": String (required),
"Email": String (required),
"Birthday": Date (optional)} 		The created JSON object as confirmation. Ex:
{"Username": String,
"Password": String,
"Email": String,
"Birthday": Date}
Allow users to update their account information. /users/:Username PUT A JSON object with keys to be updated. Ex:
{"Username": String (required),
"Password": String (required),
"Email": String (required),
"Birthday": Date (optional)} 		The updated JSON object as confirmation. Ex:
{"Username": String,
"Password": String,
"Email": String,
"Birthday": Date}
Allow users to add a movie to their list of favorites /users/:Username/movies/:MovieID POST None A JSON object reflecting the updated movie list.
Allow users to remove a movie from their list of favorites users/:Username/movies/:MovieID DELETE None A JSON object reflecting the updated movie list.
Allow existing users to deregister /users/:Username DELETE None A message confirming successful deregistration.
